The Cyber Swiss Army Knife

CyberChef is a simple, intuitive web app for carrying out all manner of "cyber" operations within a web browser. These operations include simple encoding like XOR and Base64, more complex encryption like AES, DES and Blowfish, creating binary and hexdumps, compression and decompression of data, calculating hashes and checksums, IPv6 and X.509 parsing, changing character encodings, and much more.

The tool is designed to enable both technical and non-technical analysts to manipulate data in complex ways without having to deal with complex tools or algorithms.

Running locally with Docker

Option 1: Build the Docker Image yourself

docker build --tag cyberchef --ulimit nofile=10000 .
docker run -it -p 8080:8080 cyberchef

Option 2: Use the pre-built Docker Image

docker run -it -p 8080:8080 ghcr.io/gchq/cyberchef:latest

Then navigate to http://localhost:8080 in your browser.

How it works

There are four main areas in CyberChef:

  1. The input box in the top right, where you can paste, type or drag the text or file you want to operate on.
  2. The output box in the bottom right, where the outcome of your processing will be displayed.
  3. The operations list on the far left, with all operations in categorised lists or searchable.
  4. The recipe area in the middle, where you drag operations and specify arguments.

Features

  • Drag and drop — Operations can be dragged in and out of the recipe list. Files up to 2GB can be dragged over the input box.
  • Auto Bake — CyberChef automatically bakes whenever you modify the input or recipe.
  • Automated encoding detection — Uses several techniques to automatically detect encodings.
  • Breakpoints — Set breakpoints on any operation to pause execution and step through.
  • Save and load recipes — Save recipes to local storage or copy the URL to share.
  • Entirely client-side — No data is ever sent to the CyberChef web server. Can be downloaded and run locally.

Browser support

  • Google Chrome 50+
  • Mozilla Firefox 38+
