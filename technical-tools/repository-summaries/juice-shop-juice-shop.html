README Contents

OWASP Juice Shop can be used in security trainings, awareness demos, CTFs and as a guinea pig for security tools. Juice Shop encompasses vulnerabilities from the entire OWASP Top Ten along with many other security flaws found in real-world applications.

Setup

From sources

  1. Install node.js
  2. Run git clone https://github.com/juice-shop/juice-shop.git --depth 1
  3. Go into the cloned folder with cd juice-shop
  4. Run npm install
  5. Run npm start
  6. Browse to http://localhost:3000

Docker container

  1. Install Docker
  2. Run docker pull bkimminich/juice-shop
  3. Run docker run --rm -p 127.0.0.1:3000:3000 bkimminich/juice-shop
  4. Browse to http://localhost:3000

Demo

Feel free to have a look at the latest version of OWASP Juice Shop: demo.owasp-juice.shop

Official companion guide

OWASP Juice Shop comes with an official companion guide eBook, Pwning OWASP Juice Shop, published under CC BY-NC-ND 4.0 and available for free in PDF, Kindle and ePub format on LeanPub. It gives a complete overview of all vulnerabilities including hints and full step-by-step solutions.
