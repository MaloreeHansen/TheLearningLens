What is CTFd?

CTFd is a Capture The Flag framework focusing on ease of use and customizability. It comes with everything you need to run a CTF and it's easy to customize with plugins and themes.

Features

  • Create your own challenges, categories, hints, and flags from the Admin Interface
    • Dynamic Scoring Challenges
    • Unlockable challenge support
    • Challenge plugin architecture for custom challenges
    • Static & Regex based flags with custom flag plugins
    • File uploads to the server or an Amazon S3-compatible backend
    • Automatic bruteforce protection
  • Individual and Team based competitions
  • Scoreboard with automatic tie resolution — hide or freeze scores at a specific time
  • Scoregraphs comparing the top 10 teams and team progress graphs
  • Markdown content management system
  • SMTP + Mailgun email support with confirmation and forgot password flows
  • Automatic competition starting and ending
  • Team management, hiding, and banning
  • Import and export of CTF data for archival

Install

  1. Install dependencies: pip install -r requirements.txt
  2. Modify CTFd/config.ini to your liking.
  3. Use python serve.py or flask run in a terminal to drop into debug mode.

Run with Docker:

docker run -p 8000:8000 -it ctfd/ctfd

Or with Docker Compose:

docker compose up

MajorLeagueCyber integration

CTFd integrates with MajorLeagueCyber (MLC), a CTF stats tracker providing event scheduling, team tracking, and single sign-on. Register an event, then install the client ID and secret in CTFd/config.py or the admin panel.

OAUTH_CLIENT_ID = None
OAUTH_CLIENT_SECRET = None
