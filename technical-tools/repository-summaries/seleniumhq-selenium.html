README contents

Automates browsers. That's it.

Selenium specifically provides an infrastructure for the W3C WebDriver specification — a platform and language-neutral coding interface compatible with all major web browsers.

Installing

All platforms

  • Bazelisk — a Bazel wrapper that automatically downloads the correct version.
  • Java JDK version 17 or greater. Set JAVA_HOME to the JDK location.

macOS

  • Xcode command-line tools: xcode-select --install
  • Apple Silicon: add build --host_platform=//:rosetta to .bazelrc.local

Building

Selenium uses Bazel for builds. Common commands:

bazel build //<language>/...    # build everything for a language
bazel build grid                 # build just the grid
./go <language>:build            # simplified build command

Developing

Java

IntelliJ with Bazel IJ plugin is recommended. Local Maven install:

./go java:install

Python

PEP8 style (120 char lines), enforced with ruff. Local setup:

pip install -r py/requirements.txt
./go py:local_dev

Testing

Filter tests by size or tag:

bazel test //<language>/... --test_size_filters=small
bazel test //<language>/... --test_tag_filters=chrome
LanguageCommand

Releasing

./go all:release              # build and release everything
./go <language>:release       # release a specific language
