About the list

In the field of Information Security, understanding the attacker is crucial. To hone cybersecurity skills, you need to keep hacking skills up-to-date in a safe, legal, and free environment.

This list compiles websites that offer various opportunities to practice cyber skills. Each site has a unique approach, and a brief description is provided to highlight its focus. Some sites offer tutorials to guide you, while others encourage self-directed exploration.
