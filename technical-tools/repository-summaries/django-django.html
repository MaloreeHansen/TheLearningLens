About Django

Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design. All documentation is in the docs directory and online at docs.djangoproject.com.

Getting Started

  1. Read docs/intro/install.txt for instructions on installing Django.
  2. Work through the tutorials in order (docs/intro/tutorial01.txt, docs/intro/tutorial02.txt, etc.).
  3. For deployment, read docs/howto/deployment/index.txt.
  4. Read through the topical guides in docs/topics, then jump to the HOWTOs in docs/howto for specific problems.
  5. Check the reference in docs/ref for detailed API documentation.
  6. See docs/README for instructions on building an HTML version of the docs.

Dependencies

Python (pyproject)

  • asgiref — ASGI reference implementation
  • sqlparse — SQL parser
  • tzdata — timezone data
  • argon2-cffi — Argon2 password hashing (optional)
  • bcrypt — bcrypt password hashing (optional)

JavaScript (npm)

  • @biomejs/biome — linter and formatter
  • puppeteer — headless Chrome for testing
  • grunt / grunt-cli — task runner
  • grunt-contrib-qunit / qunit — JavaScript unit tests

Django's development depends on contributions. If a project depends on Django, consider supporting the Django Software Foundation.
